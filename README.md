# QuizNest

QuizNest 是一个本地优先的中文学习与刷题工具。用户可以导入 Markdown 笔记，调用 OpenAI-compatible Chat Completions API 生成选择题和大题，并在本地完成练习、AI 判题、错题复习、薄弱点分析、解惑对话、学习统计和数据备份。

## 技术栈

- 前端：原生模块化 JavaScript，零框架依赖。
- 本地调试：Node.js 静态文件服务和 `/api/chat-completions` 代理。
- 桌面应用：Tauri 2，用于生成 macOS/Windows 本地安装包。
- 本地数据：IndexedDB 保存配置、笔记、题目、作答记录、错题、对话、统计和个人资料。

## 本地运行

```bash
npm start
```

浏览器调试地址：

```text
http://localhost:5173
```

## 常用脚本

```bash
npm run sync-dist
npm run build:dmg
npm run clean
```

- `sync-dist`：把当前前端代码同步到 Tauri 使用的 `dist/`。
- `build:dmg`：同步前端、构建 Tauri 应用、生成 `QuizNest_0.1.0_aarch64.dmg`，并清理构建缓存。
- `clean`：移除 `src-tauri/target`、临时浏览器目录和常见系统缓存文件。

## 使用流程

1. 进入“设置”，填写 Base URL、API Key、出题模型、判题模型和可选对话模型。
2. 回到首页，上传 `.md` 笔记。
3. 上传后打开生成设置，选择选择题数量、大题数量和难度。
4. 进入练习页逐题作答。选择题点击选项会立即保存，提交后锁定；大题支持文字和图片答案。
5. 错题会自动进入错题本，可筛选、删除、标记掌握、重新练习。
6. 在错题本选择具体笔记后，可以点击“薄弱点分析”。
7. 进入“解惑”，可选择笔记、题目和错题上下文进行学习与解惑。
8. 进入“统计”，查看个人资料、Token 消耗、做题热力图和模型使用情况。
9. 进入“设置”可导出或导入学习数据备份。

详细结构见 `docs/data-structure.md`。数据默认只保存在本机浏览器或 Tauri WebView 的 IndexedDB 中。

## 验收建议

- 设置页：保存配置后刷新页面，确认配置仍存在。
- 笔记页：上传带 `#`/`##` 标题的 `.md` 文件，确认章节被识别。
- 生成题目：确认生成后跳转练习页，并刷新后套题仍存在。
- 选择题：选择选项后刷新，确认选择未丢失；提交后确认题目锁定。
- 大题：输入文字或上传图片后提交，确认判题结果保存。
- 错题本：答错后确认自动出现，可筛选、删除、标记掌握和重新练习。
- 解惑：在不同页面提问，确认回答能结合已选择的学习上下文。
- 统计页：确认 Token 和做题热力图可悬停查看日期明细，个人头像和名称可保存。
- 导入导出：导出的 JSON 文件应能在 macOS/Windows 两端导入。
- LaTeX 公式：笔记和题目内容支持 `$...$`、`$$...$$`、`\(...\)`、`\[...\]`。如果 CDN 无法加载 MathJax，会保留原始 LaTeX 文本。

## 故障排查

- 如果设置页提示“本地代理无法访问外网”，通常不是 Base URL 填错，而是运行 `npm start` 的 Node 进程被沙盒、防火墙、代理或系统权限拦截。请在本机普通终端中运行 `npm start`，并确认该终端有外网访问权限。
- 如果提示“当前接口不支持 response_format=json_object”，说明该兼容服务没有实现 OpenAI 的 JSON 模式。此时可以换支持 JSON 模式的模型/服务。
- 如果提示“模型名称错误，或 Base URL 路径不兼容”，请确认 Base URL 类似 `https://example.com/v1`，模型名与服务商后台一致。
- 如果出题提示“网络超时”，请先把设置页的请求超时调到 `180000` 到 `300000` 毫秒，并减少单次生成题量。
